**Week 1 Capstone Project - Original Work**

## 🌟 Features

- ✅ Add custom habits
- ✅ Plan which habits to track with comments
- ✅ Mark habits as complete
- ✅ Calculate current streak (consecutive days)
- ✅ Show completion percentage
- ✅ Save/Load data to JSON file
- ✅ View today's habits with status

## 🚀 How to Run
